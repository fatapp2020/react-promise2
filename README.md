# Desafío: Detalle de Producto


### Clonar Proyecto Primera Vez
```
git clone https://github.com/fatapp2020/react-promise2.git
```

### Subir Proyecto
```
- git init
- git add .
- git commit -m ""
- git push -u origin main
```

### Actualizar Proyecto Último Commit
```
- git pull
```

### Levantar Proyecto
```
- npm start
```
